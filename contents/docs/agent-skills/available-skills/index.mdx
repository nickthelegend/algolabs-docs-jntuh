# Available Skills

The following skills are available to supercharge your development workflow:

| Skill | Description |
| :--- | :--- |
| `create-project` | Initialize AlgoKit projects with best practices. |
| `build-smart-contracts` | Build contracts in Algorand TypeScript (PuyaTs) or Python. |
| `test-smart-contracts` | Write and run contract tests. |
| `call-smart-contracts` | Deploy and interact with contracts. |
| `deploy-react-frontend` | Build React frontends for dApps. |
| `use-algokit-cli` | AlgoKit CLI command reference. |
| `troubleshoot-errors` | Diagnose and fix common Algorand errors. |

## x402 Protocol Skills
Skills for the x402 HTTP-native payment protocol:
- **`algorand-x402-typescript`**: Clients, servers, and facilitators in TS.
- **`algorand-x402-python`**: Clients, servers, and facilitators in Python.

## MCP Servers
Skills also provide access to Model Context Protocol (MCP) servers:
- **Kapa MCP**: Semantic search over official Algorand documentation.
- **GitHub MCP**: Access canonical examples from Algorand repositories.
