# Agent Skills Setup

Follow these steps:

1.  **Clone the Skills Repo**:
    ```bash
    git clone https://github.com/algorand-devrel/algorand-agent-skills.git
    ```
2.  **Integrate with Cursor**:
    Copy the `skills` folder into your project's `.cursor/skills` directory.
3.  **Set GITHUB_TOKEN**:
    Ensure you have a Personal Access Token set in your environment.
