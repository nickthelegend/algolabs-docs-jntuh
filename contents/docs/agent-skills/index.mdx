# Algorand Agent Skills

**Algorand Agent Skills** are specialized instructions and tools that enable AI coding assistants to understand and perform Algorand-specific development tasks.

By adding these skills to your AI (like Claude, Cursor, or Copilot), you transform it from a general-purpose assistant into an expert Algorand developer that knows:

- **Best Practices**: The right way to structure contracts and dApps.
- **Library Nuances**: Deep knowledge of AlgoKit Utils, PuyaTS, and more.
- **Common Pitfalls**: How to avoid and fix frequent errors.

## Why use Agent Skills?

1.  **Speed**: Write complex smart contracts in minutes by describing them in plain English.
2.  **Accuracy**: AI assistants loaded with skills are less likely to hallucinate incorrect API calls.
3.  **Onboarding**: New developers can become productive immediately without spending weeks reading documentation.

## How they work

Each skill consists of:
- **Instructions**: Guidelines for the AI on how to perform a task.
- **Reference Material**: API documentation and code examples that the AI can "read" on demand.
- **MCP Servers**: Tools that allow the AI to search official docs or fetch live blockchain data.
