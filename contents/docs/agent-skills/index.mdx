# Algorand Agent Skills

**Algorand Agent Skills** are specialized sets of instructions that teach AI assistants exactly how to build on Algorand.

## Why use them?

Instead of spending weeks reading documentation, you can describe your intent to your AI assistant, and it will use the Agent Skills to generate production-grade Algorand code.

Head over to the **[Setup](/docs/agent-skills/setup)** guide to get started.
