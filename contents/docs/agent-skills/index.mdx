# Algorand Agent Skills

**Algorand Agent Skills** are a "power pill" for your development workflow. They are specialized sets of instructions, tool configurations, and reference materials that teach AI assistants (like Claude, Cursor, or Copilot) exactly how to build on Algorand.

## What do they do?

When you load Agent Skills into your preferred AI tool, it gains expert knowledge in:

- **AlgoKit Workflows**: Initializing, testing, and deploying projects.
- **Smart Contract Languages**: Writing clean, efficient Puya TypeScript (PuyaTs) or Python.
- **Troubleshooting**: Diagnosing and fixing common Algorand errors instantly.
- **Standards**: Implementing ARC standards (like ARC-4, ARC-32, ARC-56) correctly.

## Why use them?

Instead of spending weeks reading documentation, you can describe your intent to your AI assistant, and it will use the Agent Skills to generate production-grade Algorand code that follows all best practices.
