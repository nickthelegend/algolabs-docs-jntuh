# Quick Start: Setting up Agent Skills

Follow these steps to integrate Algorand Agent Skills into your project.

## 1. Prerequisites
You need the **[algorand-agent-skills](https://github.com/algorand-devrel/algorand-agent-skills)** repository cloned to your machine.

```bash
git clone https://github.com/algorand-devrel/algorand-agent-skills.git
```

## 2. Setup (Per Project)

Navigate to your active project directory and copy the necessary skills and configuration files.

### For Cursor
1. Create a `.cursor/skills` directory in your project root.
2. Copy the skills:
   ```bash
   cp -r ../algorand-agent-skills/skills/* .cursor/skills/
   ```
3. Copy the MCP configuration:
   ```bash
   cp ../algorand-agent-skills/setups/cursor/mcp.json .cursor/mcp.json
   ```

### For Claude Code
```bash
# Copy the skills to your project
cp -r ../algorand-agent-skills/skills ./
# Copy the agent configuration
cp ../algorand-agent-skills/setups/claude-code/.mcp.json ./
cp ../algorand-agent-skills/setups/claude-code/CLAUDE.md ./
```

## 3. GitHub PAT Setup
All skills require access to search code examples via GitHub.
1. Create a [GitHub Personal Access Token](https://github.com/settings/personal-access-tokens).
2. Set the `GITHUB_TOKEN` environment variable:
   ```bash
   export GITHUB_TOKEN="your_token_here"
   ```

Now, your AI assistant is ready to help you build on Algorand!
