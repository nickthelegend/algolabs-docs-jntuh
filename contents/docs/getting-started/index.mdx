# Welcome to Algorand JNTU

The **Algorand JNTU** program is designed to onboard developers into the Algorand ecosystem, providing the tools and knowledge needed to build scalable, secure, and decentralized applications on the world's most advanced blockchain.

## Getting Started

To begin your journey with Algorand, we recommend watching our introductory video and setting up your development environment.

### Installation Guide
To start developing on Algorand, you'll need the following tools:

1.  **AlgoKit**: The one-stop shop for Algorand development.
    ```bash
    pipx install algokit
    ```
2.  **Docker**: Required for running a local Algorand node.
3.  **Python 3.10+** or **Node.js 18+**: Depending on your preferred language.

## Algorand Agent Skills

**Algorand Agent Skills** are a power pill for development. They teach AI assistants (like Claude, Cursor, or Copilot) how to complete Algorand-specific tasks:

- **Create Projects**: Initialize AlgoKit projects with best practices.
- **Build Smart Contracts**: Build contracts in Algorand TypeScript (PuyaTs) or Python.
- **Test & Deploy**: Automated workflows for testing and deploying your dApps.
- **Troubleshoot**: Diagnose and fix common Algorand errors instantly.

### Why use Agent Skills?
Building on blockchain can be complex. Agent skills abstract away that complexity by giving your AI pair programmer the exact context it needs to help you write production-grade code from day one.
