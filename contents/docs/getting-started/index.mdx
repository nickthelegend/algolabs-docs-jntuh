## Algorand Connect Wallet Starter

To get your frontend up and running quickly, use the **Connect Wallet Starter V2**. This template provides the easiest way to integrate Algorand wallets into your React application.

### [Connect Wallet Starter V2](https://github.com/nickthelegend/connect-wallet-starter-v2)

**Setup Instructions:**
1.  **Clone the project**:
    ```bash
    git clone https://github.com/nickthelegend/connect-wallet-starter-v2.git
    ```
2.  **Install dependencies**:
    ```bash
    cd connect-wallet-starter-v2
    npm install
    ```
3.  **Run locally**:
    ```bash
    npm run dev
    ```

## Onboarding Users without Wallets (Custodian)

One of the biggest hurdles in blockchain is user onboarding. **Kyra Wallet** (the custodian) allows users to login using **Google**—no need for Pera or any other external wallet to get started.

### [Kyra Wallet Custodian Starter](https://github.com/nickthelegend/connect-wallet-starter-v2-custodian)

**Why use the Custodian Wallet?**
-   **No Crypto Experience Needed**: Users can sign in with social accounts they already have.
-   **Seamless Integration**: The wallet is generated automatically in [kyra.algocraft.fun](https://kyra.algocraft.fun).
-   **Developer Friendly**: Use the starter template to build apps that onboard users instantly.

## Algorand Agent Skills

**Algorand Agent Skills** are a power pill for development. They teach AI assistants (like Claude, Cursor, or Copilot) how to complete Algorand-specific tasks.

- **Create Projects**: Initialize AlgoKit projects with best practices.
- **Build Smart Contracts**: Build contracts in Algorand TypeScript (PuyaTs) or Python.
- **Test & Deploy**: Automated workflows for testing and deploying your dApps.
- **Troubleshoot**: Diagnose and fix common Algorand errors instantly.
