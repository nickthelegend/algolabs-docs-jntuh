# Getting Started with Algorand

New to Algorand? This page will guide you through the essential resources and point you in the right direction to start building.

## Where to Start?

The Algorand ecosystem is rich with resources. Here are the most important entry points:

1.  **[Algorand Developer Portal](https://developer.algorand.org/)**: The source of truth for all technical documentation, API references, and in-depth tutorials.
2.  **[AlgoKit](https://github.com/algorandfoundation/algokit-cli)**: The official development toolkit. If you're building on Algorand, you'll be using AlgoKit.
3.  **[Discord Community](https://discord.gg/algorand)**: Join thousands of other developers to ask questions and get support.

## Your Path to Development

Your journey typically follows these steps:

1.  **Understand the Basics**: Learn about Accounts, Assets (ASAs), and Transactions.
2.  **Install the Tooling**: Set up your development environment.
3.  **Write your First Contract**: Use Python or TypeScript to logic on-chain.
4.  **Connect a Frontend**: Build a user interface that interacts with your contracts.
