# Getting Started

To begin your journey with Algorand, we recommend the following path:

1.  **Understand the Basics**: Learn about Accounts, Assets (ASAs), and Transactions on the [Algorand Developer Portal](https://developer.algorand.org/).
2.  **Environment Setup**: Install the necessary tools to build locally.
3.  **Watch the Tutorial**: See a live demonstration of the workflow.
