# Installation

To build on Algorand, you'll need the following core tools:

### 1. Python 3.10+
Most Algorand tooling, including AlgoKit, is built with Python.
- [Download Python](https://www.python.org/downloads/)

### 2. Docker Desktop
Required for running an Algorand "LocalNet" (a private network on your machine).
- [Install Docker](https://www.docker.com/products/docker-desktop/)

### 3. AlgoKit CLI
AlgoKit is the primary interface for starting projects.

```bash
# Install via pip
pipx install algokit

# Verify
algokit --version
```
