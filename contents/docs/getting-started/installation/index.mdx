## What to Install

To build on Algorand, you'll need the following core tools:

### 1. Python 3.10+
Most Algorand tooling, including AlgoKit and Puya Py, is built with Python.
- [Download Python](https://www.python.org/downloads/)

### 2. Docker Desktop
Required for running an Algorand "LocalNet" (a private private network on your machine).
- [Install Docker](https://www.docker.com/products/docker-desktop/)

### 3. AlgoKit CLI
AlgoKit is the primary interface for starting projects, running local nodes, and deploying contracts.

```bash
# Install via pipx (recommended)
pipx install algokit

# Verify installation
algokit --version
```

### 4. Node.js (Optional but Recommended)
Required if you are building React/Next.js frontends or using the TypeScript SDK.
- [Install Node.js](https://nodejs.org/)

---

## Spin up your First Node
Once Docker and AlgoKit are installed, you can start a local Algorand node:

```bash
algokit localnet start
```

This gives you a functional Algorand network with pre-funded accounts to test your applications for free.
