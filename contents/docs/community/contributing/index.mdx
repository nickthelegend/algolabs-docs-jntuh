# Joining the Community

We welcome contributions from developers, instructors, and advocates! Whether it's adding a new agent skill or creating tutorial content, your input is valuable.

1. **Fork the Repo**: Start by forking the relevant project on GitHub.
2. **Create a Branch**: Work on your feature or fix in a dedicated branch.
3. **Submit a PR**: Provide a clear description of your changes.

Check out our [GitHub Discussions](https://github.com/algorand-devrel) for any major proposals.
