# Starters & Templates

Jumpstart your development with these high-quality templates for frontend and user onboarding.

## 1. Connect Wallet Starter V2
The definitive starter for building React/Next.js applications on Algorand. It includes the logic for connecting wallets like Pera, Defly, and Kibisis out of the box.

### [Download on GitHub](https://github.com/nickthelegend/connect-wallet-starter-v2)

**Setup Steps:**
1.  Clone: `git clone https://github.com/nickthelegend/connect-wallet-starter-v2.git`
2.  Install: `npm install`
3.  Run: `npm run dev`

---

## 2. Kyra Custodian Wallet (Onboarding)
Perfect for onboarding users who have never touched blockchain. This template allows users to sign in with **Google**—no Pera wallet required to start.

### [Custodian Starter on GitHub](https://github.com/nickthelegend/connect-wallet-starter-v2-custodian)

**Why use Kyra?**
-   **Zero Friction**: Users get a functional Algorand wallet just by logging in with their Google account.
-   **KYRA Ecosystem**: The wallet is securely managed at [kyra.algocraft.fun](https://kyra.algocraft.fun).
-   **Mass Adoption**: Ideal for building web3 apps with a traditional web2 user experience.
