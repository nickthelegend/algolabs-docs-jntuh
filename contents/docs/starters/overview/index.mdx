# Starters & Templates

Jumpstart your development with these pre-configured templates.

## 1. Connect Wallet Starter V2
The easiest way to integrate Algorand wallets into your React application. Supports Pera, Defly, and other major wallets out of the box.

- **GitHub**: [connect-wallet-starter-v2](https://github.com/nickthelegend/connect-wallet-starter-v2)
- **Features**: 
    - Wallet Connection UI
    - Transaction Signing examples
    - Account balance fetching

## 2. Custodian Wallet Starter
Specifically designed for onboarding users who don't have a blockchain wallet yet. Allows users to login via **Google**.

- **GitHub**: [connect-wallet-starter-v2-custodian](https://github.com/nickthelegend/connect-wallet-starter-v2-custodian)
- **Live Demo**: [kyra.algocraft.fun](https://kyra.algocraft.fun)
- **Key Benefit**: Zero-friction onboarding for non-crypto users.

## 3. Deployment Guide

Most starters can be deployed instantly to **Vercel** or **Netlify**:

1. Push your code to GitHub.
2. Connect your repo to Vercel/Netlify.
3. Add any required environment variables (like `NEXT_PUBLIC_ALGOD_SERVER`).
4. Build and Deploy!
